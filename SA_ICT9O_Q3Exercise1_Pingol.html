HTML

HyperText Markup Language

It is a system for tagging text. It can contain fonts, colors, images, and hyperlinks. This can be seen on web pages.

CSS

Cascading Style Sheets

It is a style sheet language that is used to describe how a webpage will look. It makes designing a webpage easier. It can either be put at the start of the codes or in a seperate file.

It helps load a program. It uses a few changes in code. Bootstrap is mostly used to create websites or applications.
